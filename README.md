# TRANSPORTES JC

Sitio web de la empresa de transporte, fletes y mudanzas **TRANSPORTES JC**.

- **Desarrollo local:** aplicación [Laravel](https://laravel.com); las vistas viven en `resources/views/`.
- **GitHub Pages:** GitHub no ejecuta PHP. La versión pública estática se publica desde la carpeta [`docs/`](./docs/) (HTML, CSS, JS e imágenes). Tras cambiar estilos o textos en Laravel, copia de nuevo `public/css`, `public/js` y `public/images` a `docs/` o ajusta los HTML en `docs/` para que coincidan.

## Sitio en GitHub Pages

Configura el repositorio en **Settings → Pages → Build and deployment**: origen **Deploy from a branch**, rama **main**, carpeta **`/docs`**.
